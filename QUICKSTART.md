# Image Downloader - Quick Start Guide

Get up and running in 3 simple steps!

---

## Step 1: Load the Extension

1. Open Chrome and go to `chrome://extensions/`
2. Turn ON **Developer mode** (top-right toggle)
3. Click **Load unpacked** and select the `downloader` folder

---

## Step 2: Configure Settings (Optional)

Click the extension icon in your toolbar:

| Setting | Default | What it does |
|---------|---------|--------------|
| Enable Downloads | ON | Turn auto-download on/off |
| Minimum Size | 600px | Only download images larger than this |
| Destination Folder | `downloaded-images` | Where to save (in Downloads) |

---

## Step 3: Start Browsing!

Visit any website with images. Large images will automatically download to:

```
Downloads/downloaded-images/img-1.png
Downloads/downloaded-images/img-2.jpg
...
```

---
