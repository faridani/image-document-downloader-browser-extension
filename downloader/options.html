How It Works

  • Enable the extension to start monitoring images as you browse
  • Images larger than the specified size will be automatically downloaded
  • Downloaded images are saved with sequential names: img-1.png, img-2.jpg, etc.
  • The counter increments with each download and persists across browser sessions
  • Use the Reset Counter button to start numbering from 1 again
