# Quick Start Guide

## Get Started in 3 Steps!

### Step 1: Install the Extension
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right)
3. Click "Load unpacked" and select the `downloader` folder

### Step 2: Configure Settings
1. Click the extension icon in your toolbar
2. Toggle "Enable Auto Download" to ON
3. Click "Save Settings"

### Step 3: Start Browsing!
Visit any website with images. Images larger than 600px will automatically download to:
```
Downloads/downloaded-images/img-1.png
Downloads/downloaded-images/img-2.jpg
Downloads/downloaded-images/img-3.webp
...
```

## Default Settings
- **Enabled**: OFF (must be turned ON)
- **Minimum Size**: 600 pixels
- **Destination Folder**: "downloaded-images"
- **Starting Counter**: 1

## Customization Tips
- **Increase size threshold** (e.g., 1000px) to download only very large images
- **Change folder name** to organize different browsing sessions
- **Reset counter** when starting a new collection
- **Disable when not needed** to avoid unwanted downloads

## Testing
Open `test.html` in Chrome to see the extension in action with sample images.
